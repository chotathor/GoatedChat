## Download
**[Get the APK](https://github.com/chotathor/GoatedChat/releases)**

## Features
- BLE mesh networking (multi-hop relay, up to 7 hops)
- Noise Protocol end-to-end encryption
- Nostr protocol for global bridge (optional)
- Location-based geohash channels
- IRC-style commands (/msg, /who, /slap)
- Emergency wipe + ephemeral keys
- LZ4 message compression

## How It Works
Creates a decentralized mesh network via Bluetooth. Messages hop through nearby devices. Optional Nostr bridge syncs globally.

## Tech
Kotlin • Jetpack Compose • Nordic BLE • Noise Protocol • Nostr • Tor (Arti)
